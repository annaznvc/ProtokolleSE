# Protokoll 3

## Aufgabenstellung

- Take the Classes from your domain model
- Start developing them in Scala
- Use Behaviour Driven Development
- Use ScalaTest
- Write a Spec using WordSpec 
- Use the IDEA ScalaTest Runner 
- Achieve 100% code coverage. And maintain it!
